# SQL LeetCode Practice

This repository contains my daily SQL practice, mainly from LeetCode. I’m using it to improve how I think through data problems, write efficient queries, and get more comfortable with real-world scenarios.

---

## About

The goal of this repository is consistency and growth. Instead of just solving problems, I’m focusing on understanding patterns,improving query structure, and building a strong foundation for data roles.

---

## What I’m Practicing

* Query fundamentals (SELECT, WHERE)
* Aggregations and filtering (GROUP BY, HAVING)
* Joins (INNER, LEFT, RIGHT, FULL)
* Subqueries and Common Table Expressions (CTEs)
* Window functions (ROW_NUMBER, RANK, etc.)
* Data cleaning and transformation
* Case statements and conditional logic
* More advanced SQL patterns over time
